# To-Do-List

## Bu repo [Patica.dev](https://www.patika.dev/tr) JavaScript dersinde öğrendiğimiz bilgileri pekiştirmek amaçlı oluşturduğum To do list projesi reposudur. İçerisinde birer adet README, Css, Html ve Js dosyası barındırmaktadır.

---

<br/>

## _Aşağıda To Do List uygulamasının boş halini görmektesiniz. Uygulamaya yapılacak birşey eklemek için "Add something to do list..." yazan kısma yapılacak şeyi yazıp yandaki yeşil buton yardımı ile listeye ekleyebilirsiniz._
